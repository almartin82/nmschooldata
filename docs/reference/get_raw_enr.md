# Download raw enrollment data from NM PED

Downloads the 40-Day enrollment report Excel file from NM PED's website.
NM PED publishes enrollment counts annually with breakdowns by
demographics.

## Usage

``` r
get_raw_enr(end_year)
```

## Arguments

- end_year:

  School year end (2023-24 = 2024)

## Value

Data frame with enrollment data
