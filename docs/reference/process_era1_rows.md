# Process Era 1 rows into standard format

Process Era 1 rows into standard format

## Usage

``` r
process_era1_rows(df, end_year, row_type, find_col)
```

## Arguments

- df:

  Data frame of rows to process

- end_year:

  School year end

- row_type:

  "School" or "District"

- find_col:

  Function to find columns by pattern

## Value

Processed data frame
